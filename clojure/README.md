#### Installation and REPL usage


``` clojure

;; clojure installation (on mac)
brew install leiningen

;; REPL
lein repl

;; show exception
(pst)

;; loading a file in REPL
(load-file "basic_repl.clj")

;; create new project
lein new app <project-name-here>

;; run the project
lein run

;; create a JAR file
lein uberjar

```
